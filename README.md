# 📊 HR Analytics Dashboard | Power BI + Oracle SQL

An interactive HR Analytics Dashboard built using **Power BI** and **Oracle SQL** to analyze employee attrition, workforce demographics, salary distribution, and employee performance metrics.

This project demonstrates end-to-end data analytics skills, including SQL data analysis, dashboard development, and business insight generation.

---

## 📌 Project Overview

The objective of this project is to help HR teams monitor workforce statistics and identify the major factors contributing to employee attrition.

The dashboard enables decision-makers to:

- Monitor employee attrition
- Analyze workforce demographics
- Compare salary across job roles
- Evaluate employee distribution
- Identify departments with high attrition
- Support HR decision-making using data

## 📊 Key Business Insights

- Sales department has one of the highest employee attrition rates.
- Employees working overtime are more likely to leave the organization.
- Monthly income varies significantly across job roles.
- Employee demographics provide valuable workforce insights.
- HR managers can use these findings to improve employee retention strategies.
